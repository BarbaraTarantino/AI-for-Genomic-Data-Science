# AI & Machine Learning for Genomic Data Science

Teaching materials for the Master’s course held at the **University of Pavia**.  
This repository includes slides and interactive Colab notebooks across five days of lectures and tutorials on modern AI and ML techniques for biomedical and genomic data.

## Learning Outcomes
By the end of the course, participants will be able to:
- Use **Python and scientific libraries** for genomic data analysis.  
- Apply **supervised and unsupervised ML algorithms** to biomedical datasets.  
- Implement and train **deep neural networks** using PyTorch.  
- Apply **computer vision techniques** to medical imaging data.  
- Understand and experiment with **Large Language Models** for clinical text.  
- Adopt **reproducible and transparent workflows** for biomedical AI research.

---

## Datasets
The tutorials include public datasets such as:
- Colon Cancer (Alon et al., 1999)  
- Breast Cancer gene expression data  
- PathMNIST histology images  
- Biomedical text corpora for LLM examples
